        Skills

        Upload zip packages that contain only one SKILL.md file. Skill packages stay local to this browser in dedicated skills storage. Packages with extra files can still be reviewed here, but only single-file SKILL.md packages are exposed to the model.

        Uploaded skill packages

        Browse the stored SKILL.md content for each package or remove a package from this browser.

        Proxy

        Uses a prefix-style proxy only when a direct browser request appears blocked by CORS. Proxy URLs must use https, or http://localhost, must not include credentials or fragments, may include query-string prefixes such as ?url=, and are tested with an MCP initialize probe against https://example-server.modelcontextprotocol.io/mcp before saving.

        MCP Servers

        Only browser-reachable MCP HTTP endpoints without OAuth or token authentication are supported.

        Configured MCP servers

        Imported servers and commands stay local to this browser. Enable the server and the specific commands you want exposed to the model.
